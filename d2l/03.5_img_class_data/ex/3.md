# 3.查阅框架的在线API文档。还有哪些其他数据集可用？

PyTorch官网主要提供的是各领域库（torchvision, torchtext, torchaudio）的**API文档**，并不直接维护一个统合的数据集目录。不过，要查找更多的数据集，可以从下面这些渠道入手。

### 📚 PyTorch生态内置数据集
PyTorch在其生态的不同领域库中内置了丰富的数据集，方便快速上手和实验。
*   **图像与视频**：主要来自 `torchvision.datasets` 模块，包含**70多个**数据集，涵盖图像分类、目标检测、语义/实例分割、视频分类等领域。
*   **文本与自然语言处理 (NLP)**：主要来自 `torchtext.datasets` 模块，提供约**30个**数据集，涵盖文本分类、语言建模、机器翻译、序列标注、问答系统等多种NLP任务。
*   **音频与信号处理**：主要来自 `torchaudio.datasets` 模块，提供多种音频数据集，涵盖语音识别、语音合成、音乐信息检索、关键词唤醒等语音任务。

### 🚀 更广泛的数据集渠道
除了框架内置的数据集，海量的外部资源也值得探索。
*   **`torchdata` 项目**：这是PyTorch官方推出的**数据加载和预处理库**，它提供的 `DataPipe` 功能可以帮助你从HuggingFace Datasets、TensorFlow Datasets等标准来源高效地接入数据。
*   **跨框架数据集库**：许多知名的开源库专注于提供跨框架的数据集访问。
    *   **Hugging Face Datasets**：一个覆盖面极广的库，包含数千个 NLP、计算机视觉和音频数据集，可在项目根目录设置 `cache_dir` 参数来管理下载路径。
    *   **TensorFlow Datasets (TFDS)**：与Hugging Face Datasets类似，也提供了数百个跨领域的数据集，同样支持将数据直接加载为 `numpy` 数组或自定义的 `Dataset` 对象。
*   **其他第三方库**：针对特定的研究领域，也有一些专门的第三方库。
    *   `PyG-Hyper-Data`：用于超图学习的库。
    *   `torchdatasets`：一个提供PyTorch额外数据集的库。
*   **独立数据集平台**：如果上述来源仍然无法满足需求，可以直接到**Kaggle**、**Papers with Code**、**Google Dataset Search**等大型数据竞赛和托管平台搜索特定领域的数据集。
*   **创建自定义数据集**：对于所有项目独特的数据源，实现一个自定义的 `torch.utils.data.Dataset` 类来加载和处理数据是最根本的解决方案。
