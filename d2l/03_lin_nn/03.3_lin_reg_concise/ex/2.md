## 3. 给出的 Huber 损失公式与标准实现的对应

题目给出的公式为：

$$
l(y, y') = 
\begin{cases} 
|y - y'| - \dfrac{\sigma}{2}, & |y - y'| > \sigma \\[6pt]
\dfrac{1}{2\sigma}(y - y')^2, & \text{其它情况}
\end{cases}
\tag{3.3.1}
$$

**标准 Huber 损失**通常写作（以 PyTorch 的 `HuberLoss(delta=δ)` 为例）：
$$
L_\delta(e)=
\begin{cases}
0.5\, e^2, & |e| \le \delta \\
\delta\,(|e| - 0.5\,\delta), & |e| > \delta
\end{cases}
$$

将标准形式整体除以 `δ`，并令 `δ = σ`，就恰好得到公式 (3.3.1)：
$$
\frac{L_\sigma(e)}{\sigma}=
\begin{cases}
\frac{1}{2\sigma} e^2, & |e| \le \sigma \\
|e| - \frac{\sigma}{2}, & |e| > \sigma
\end{cases}
$$

因此：
- **公式 (3.3.1) = 标准 Huber 损失 / σ**
- 在代码中可直接使用框架内置的 Huber 损失，然后除以 σ，或自定义函数。

## 4. 用 Huber 损失代替原损失（以 PyTorch 为例）

假设原始训练代码使用 **均方误差**：
```python
loss_fn = nn.MSELoss()          # 原损失
```

现在将它替换为公式 (3.3.1) 定义的 Huber 损失，阈值 σ 可设为 1.0（或根据数据调整）。

### 方法一：直接使用 `nn.HuberLoss` 并除以 σ
```python
sigma = 1.0
huber = nn.HuberLoss(delta=sigma, reduction='none')  # 逐元素计算

def huber_scaled(pred, target):
    return huber(pred, target).mean() / sigma
```
- 这里 `HuberLoss(delta=sigma)` 计算标准 Huber 损失，`reduction='none'` 返回逐元素损失，除以 σ 后再求平均。

### 方法二：自定义函数（完全按公式实现）
```python
def huber_loss(pred, target, sigma=1.0):
    error = pred - target
    abs_error = torch.abs(error)
    quadratic = 0.5 / sigma * (error ** 2)
    linear = abs_error - 0.5 * sigma
    return torch.where(abs_error > sigma, linear, quadratic).mean()
```

### 方法三：使用 `torch.nn.SmoothL1Loss` 配合缩放
PyTorch 的 `SmoothL1Loss` 在 `beta=σ` 时等价于标准 Huber 损失除以 σ，因此：
```python
loss_fn = nn.SmoothL1Loss(beta=sigma)   # 直接得到公式 (3.3.1) 的损失
```
这是最简洁的写法，因为 `SmoothL1Loss` 的定义正是：
$$
l(e)=
\begin{cases}
0.5\, e^2/\beta, & |e| < \beta \\
|e| - 0.5\,\beta, & \text{其它}
\end{cases}
$$
与公式完全一致。

## 5. 完整替换示例（训练循环片段）

```python
import torch
import torch.nn as nn

sigma = 1.0

model = YourModel()
# 原损失: criterion = nn.MSELoss()
criterion = nn.SmoothL1Loss(beta=sigma)   # 用 (3.3.1) 的 Huber 损失

optimizer = torch.optim.SGD(model.parameters(), lr=0.01)

for inputs, targets in dataloader:
    optimizer.zero_grad()
    outputs = model(inputs)
    loss = criterion(outputs, targets)
    loss.backward()
    optimizer.step()
```

## 6. 小结

- 主流框架提供了丰富的损失函数（MSE、L1、Huber 等）和参数初始化方法（Xavier、He 等）。
- 题目中的 Huber 形式等于标准 Huber 损失除以 σ，在 PyTorch 中可直接用 `SmoothL1Loss(beta=σ)` 实现，无需额外处理。
- 将 MSE 替换为 Huber 损失后，模型对异常值的敏感度降低，训练更加稳健。

如果需要在 TensorFlow/Keras 中实现，可使用 `tf.keras.losses.Huber(delta=sigma)` 然后除以 `sigma`，或自定义损失函数，思路完全一致。
